# 🎬 MoviesHub

A modern and responsive **Movie Discovery Web App** built using **React.js**.  
Users can explore trending movies, search films, and view details in a clean UI.

---

## 🔥 Features

- 🎥 Browse popular & trending movies
- 🔍 Search movies in real-time
- 🧾 Movie cards with posters & titles
- 🎟️ "Book Now" UI (demo feature)
- 📄 About Page with project details
- 📞 Contact Page
- 📱 Fully Responsive Design
- ⚡ Fast performance using React

---

## 🛠️ Tech Stack

- ⚛️ React.js
- 🎨 CSS / Bootstrap
- 🌐 Movie API (like TMDB)
- 📦 Axios / Fetch API
